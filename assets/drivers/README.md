# Driver Pixel Portrait Sprite Sheet Spec

Each driver portrait is a PNG sprite sheet generated by `tools/generate_driver_sprites.py`.

- Frame size: `128x160`
- Sheet size: `512x1920`
- Grid: `4 columns x 12 rows`
- Palette target: 16-32 visible colors per driver
- Render scale in the mini program: `384rpx x 480rpx`

Rows are grouped by expression, with three viewing directions per expression:

1. `idle/front`
2. `idle/left`
3. `idle/right`
4. `eat/front`
5. `eat/left`
6. `eat/right`
7. `celebrate/front`
8. `celebrate/left`
9. `celebrate/right`
10. `tired/front`
11. `tired/left`
12. `tired/right`

Columns are four micro-animation frames. Current loops support blink, body breathing,
mouth changes, food props, celebration hand motion, and tired posture.

The generator also exports individual runtime frames under `assets/drivers/frames`.
The mini program displays those single-frame PNGs for stability, while the full
sprite sheet remains the source-of-truth delivery asset.

Current main roster:

verstappen.png  Red Bull      deep blue + red
leclerc.png     Ferrari       red + yellow/white
hamilton.png    Ferrari       red + white/purple accent

The portraits are stylized pixel reinterpretations. Use official F1 or team portraits
only as likeness reference; do not shrink or trace photos directly into production art.
